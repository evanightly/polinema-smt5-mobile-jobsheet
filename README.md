## Struktur Repositori

Repositori ini memiliki struktur berikut:

- `master`: Branch utama yang berisi informasi umum tentang repositori ini dan daftar isi.
- `bab1`: Instalasi Dan Pengenalan Flutter
- `bab2`: Basic Aplikasi Flutter
- `bab3`: Aplikasi Dengan Statefull Widget
- `bab4`: Statefull Widget Dan Map
- `bab5`: Navigasi Dan Rute
- `bab6`: Database Sqlite
- `bab7`: Http Request
- `bab8`: Firebase

Setiap branch bab akan berisi hasil tugas berbentuk source code project flutter
